# womenintech.now.sh

## Running Project Locally

To run the project locally you need to have Nodejs and npm installed.

1. Install the Project Dependencies

```bash
  npm install
```

2. Run React

```bash
  npm start
```
3. Local Host Server

```bash
  you will get the local host server in http://localhost:3000/
```
